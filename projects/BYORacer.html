About Game

BYORacer immerses players in a captivating sci-fi, post-apocalyptic world where they engage in thrilling competition using their BYOCraft and Apostle NFTs. It's a game that blends strategy with racing as players customize their BYOCrafts' stats based on their hard-earned experience points.

The game's inaugural phase, "The Absorption Experiment," introduces a groundbreaking survival time trial mode. This mode seamlessly combines the need for speed with the imperative to survive, all while competing against the ghosts of top players, creating an intense and competitive atmosphere.


Technical contributions

  • Implementation of the camera system using Cinemachine and multiple custom techniques.

  • Editing and integration of VFX to technical requirements that enhance the game's atmosphere.

  • API integration for blockchain-based NFT assets.

  • Core gameplay mechanics such as the time trial system, respawn system, rewards system, and custom physics across different instances of the game.

  • Character/pilot animations for the racing environment, as well as their integration in the pertinent states.

  • Testing and bug fixing across multiple various aspects of the project.

  • Scalable tool development in order to make the development process faster.
